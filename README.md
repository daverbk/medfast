# Medfast Backend

Backend repository for MedFast project

## Getting Started Locally

### Prerequisites

- Java 17 or higher

### Installation

1. Clone the repository
```bash
git clone https://github.com/JavaInt2024/medfast-be.git
```

2. Build the project
```bash
./gradlew build
```

3. Run the project
```bash
./gradlew bootRun
```
